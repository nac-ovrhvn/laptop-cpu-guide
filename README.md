Choosing the right processor for your device is a crucial task.<br>
CPU manufacturers usually have extremely confusing or recursive naming conventions that fails to show the real capabilities of the chip.<br>

This interactive tool was made to get away from the boring performance graphs and scores and explain with respect to real world tasks and requirements.<br>
